# jms_install
Jumpserver 安装脚本

官方安装文档 http://docs.jumpserver.org

Use:

```
cd /opt
git clont --depth=1 https://github.com/wojiushixiaobai/jms_install.git
cd jms_install
cp config_example.conf config.conf
vi config.conf
./jmsctl -h
```
Install
```
sh jmsctl install
```
Uninstall
```
sh jmsctl uninstall
```
Help
```
sh jmsctl -h
```
